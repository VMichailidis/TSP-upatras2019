Εγκατάσταση της Python
Αρχικά πρέπει να εγκατασταθεί η έκδοση 3.7.5 της Python, (η επιλογή της εν λόγω έκδοσης γίνεται με κριτήριο την προσφερόμενη σταθερότητα και προς αποφυγήν δυσκολιών εγκατάστασης). Η εγκατάσταση γίνεται μέσω του συνδέσμου: https://www.python.org/downloads/release/python-375/ .
Μετά την λήψη του executable file επιλέγουμε την έναρξη της εγκατάστασης.
Επιλέγουμε το ‘Add Python 3.7 to PATH’ και συνεχίζουμε με την εντολή ‘Customize installation’.
Αφού σιγουρευτούμε ότι όλες οι επιλογές έχουν επιλεχθεί (Documentation = true, pip = true, tcl/tk and IDLE = true, Python test suite = true, py launcher = true, for all users = true) πατάμε την επιλογή ‘Next’.
Στην συνέχεια αφού επιλέξουμε τις επιλογές (Associate files with python = true, add python to enviroment variables = true)πατάμε το ‘Install’.
Με τον τις προαναφερθείσες εντολές έχουμε δημιουργήσει ένα νέο Path στον υπολογιστή μας, απαραίτητο για την συνέχεια. Προσοχή! Δεν πρέπει να υπάρχει άλλο Path της Python, όποτε σε περίπτωση που προϋπήρχε θα πρέπει να γίνει σωστό uninstall της προηγούμενης έκδοσης της Python μέσω των Windows και εκκαθάριση τυχών εναπομεινάντων Path.
Εγκατάσταση των βιβλιοθηκών ‘Pymprog’ και ‘swiglpk’
Τώρα θα πρέπει να γίνει εγκατάσταση των βιβλιοθηκών οι οποίες χρησιμοποιήθηκαν.
Ανοίγουμε μια γραμμή εντολών (Command Prompt).
Κατόπιν πληκτρολογούμε την  εντολή ‘pip install swiglpk’
Η εγκατάσταση ολοκληρώνεται.
Σε περίπτωση που ζητηθεί update της ‘pip install’ ,
τότε πληκτρολογούμε την εντολή ‘python –m pip install –upgrade pip
Στην συνέχεια εκτελούμε την εντολή ‘pip install pymprog’
